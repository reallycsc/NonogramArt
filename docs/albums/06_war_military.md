| 属性 | 内容 |
| ---- | ---- |
| 序号 | 06 |
| 书架 | 人文历史 |
| 图片数 | 100 |
| 分块数/图 | 6 |
| 谜题数 | 620（常规600 + 专家20） |
| 难度配置 | 10×10~25×25 |
| 参考著作 | 富勒《西洋世界军事史》 |

## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前28张图片，入门难度
- **15×15难度**：画册中间24张图片，中等难度挑战
- **20×20难度**：画册后期28张图片，较高难度
- **25×25难度**：画册末尾20张图片，专家难度

### 难度递进示意

```
10×10 (28张) → 15×15 (24张) → 20×20 (28张) → 25×25 (20张)
第1-28张          第29-52张          第53-80张         第81-100张
```
