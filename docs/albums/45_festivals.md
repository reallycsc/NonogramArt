## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前18张图片，入门难度
- **15×15难度**：画册中间15张图片，中等难度挑战
- **20×20难度**：画册后期19张图片，较高难度
- **25×25难度**：画册末尾13张图片，专家难度

### 难度递进示意

```
10×10 (18张) → 15×15 (15张) → 20×20 (19张) → 25×25 (13张)
第1-18张          第19-33张          第34-52张         第53-65张
```
