## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前21张图片，入门难度
- **15×15难度**：画册中间18张图片，中等难度挑战
- **20×20难度**：画册后期21张图片，较高难度
- **25×25难度**：画册末尾15张图片，专家难度

### 难度递进示意

```
10×10 (21张) → 15×15 (18张) → 20×20 (21张) → 25×25 (15张)
第1-21张          第22-39张          第40-60张         第61-75张
```

**像素图：**
- 命名规范：原图文件名 + `_pixel.jpg`
- 示例：`sculpture_000_pixel.jpg`

- 目录：`data/puzzles/sculpture/`
- 常规关卡命名：`{图片ID}_{分块索引}.json`

- 专家关卡命名：`{图片ID}_expert.json`
