## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前16张图片，入门难度
- **15×15难度**：画册中间14张图片，中等难度挑战
- **20×20难度**：画册后期18张图片，较高难度
- **25×25难度**：画册末尾12张图片，专家难度

### 难度递进示意

```
10×10 (16张) → 15×15 (14张) → 20×20 (18张) → 25×25 (12张)
第1-16张          第17-30张          第31-48张         第49-60张
```

**图片规格：**
- 分辨率：2496×1664（3:2宽高比）
- 分块方式：3×2 = 6块/图
- 常规谜题数：360个（每张图6个分块谜题）
- 25×25专家谜题数：12个（第49-60号图片各1个整图谜题）
- 谜题总数：372个

**像素图：**
- 命名规范：原图文件名 + `_pixel.jpg`
- 示例：`textures_000_pixel.jpg`

**关卡文件：**
- 目录：`data/puzzles/textures/`
- 常规关卡命名：`{图片ID}_{分块索引}.json`
  - 示例：`textures_000_0.json` ~ `textures_000_5.json`
- 专家关卡命名：`{图片ID}_expert.json`
  - 示例：`textures_048_expert.json`
  - 仅第49-60号图片包含专家关卡
