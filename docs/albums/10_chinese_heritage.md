## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前23张图片，入门难度
- **15×15难度**：画册中间20张图片，中等难度挑战
- **20×20难度**：画册后期25张图片，较高难度
- **25×25难度**：画册末尾17张图片，专家难度

### 难度递进示意

```
10×10 (23张) → 15×15 (20张) → 20×20 (25张) → 25×25 (17张)
第1-23张          第24-43张          第44-68张         第69-85张
```
