## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前22张图片，入门难度
- **15×15难度**：画册中间19张图片，中等难度挑战
- **20×20难度**：画册后期23张图片，较高难度
- **25×25难度**：画册末尾16张图片，专家难度

### 难度递进示意

```
10×10 (22张) → 15×15 (19张) → 20×20 (23张) → 25×25 (16张)
第1-22张          第23-41张          第42-64张         第65-80张
```
