## 难度设计说明

本画册为新手引导画册，每张图片对应4个动态难度的数织关卡。难度只有5×5、10×10、15×15三档。文档中仅设定最低和最高难度，实际关卡由动态生成系统根据难度范围自动生成4个递进难度关卡。

### 动态难度策略

| 章节 | 难度范围 | 说明 |
| ---- | -------- | ---- |
| 第1章：萌宠乐园 | 5×5～10×10 | 前期关卡，从5×5开始逐步提升到10×10 |
| 第2章：农场伙伴 | 5×5～10×10 | 前期关卡，从5×5开始逐步提升到10×10 |
| 第3章：森林精灵 | 5×5～15×15 | 后期关卡，从5×5开始逐步提升到15×15 |
| 第4章：海洋与天空 | 5×5～15×15 | 后期关卡，从5×5开始逐步提升到15×15 |
