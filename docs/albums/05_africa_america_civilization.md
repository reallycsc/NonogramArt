## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前25张图片，入门难度
- **15×15难度**：画册中间21张图片，中等难度挑战
- **20×20难度**：画册后期26张图片，较高难度
- **25×25难度**：画册末尾18张图片，专家难度

### 难度递进示意

```
10×10 (25张) → 15×15 (21张) → 20×20 (26张) → 25×25 (18张)
第1-25张          第26-46张          第47-72张         第73-90张
```
