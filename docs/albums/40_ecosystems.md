## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前16张图片，入门难度
- **15×15难度**：画册中间14张图片，中等难度挑战
- **20×20难度**：画册后期18张图片，较高难度
- **25×25难度**：画册末尾12张图片，专家难度

### 难度递进示意

```
10×10 (16张) → 15×15 (14张) → 20×20 (18张) → 25×25 (12张)
第1-16张          第17-30张          第31-48张         第49-60张
```
