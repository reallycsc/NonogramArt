## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前19张图片，入门难度
- **15×15难度**：画册中间16张图片，中等难度挑战
- **20×20难度**：画册后期21张图片，较高难度
- **25×25难度**：画册末尾14张图片，专家难度

### 难度递进示意

```
10×10 (19张) → 15×15 (16张) → 20×20 (21张) → 25×25 (14张)
第1-19张          第20-35张          第36-56张         第57-70张
```

**像素图：**
- 命名规范：原图文件名 + `_pixel.jpg`
- 示例：`photography_000_pixel.jpg`

**关卡文件：**
- 目录：`data/puzzles/photography/`
- 常规关卡命名：`{图片ID}_{分块索引}.json`
  - 示例：`photography_000_0.json` ~ `photography_000_5.json`
- 专家关卡命名：`{图片ID}_expert.json`
  - 示例：`photography_056_expert.json`
  - 仅第57-70号图片包含专家关卡
