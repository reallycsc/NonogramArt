## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前21张图片，入门难度
- **15×15难度**：画册中间18张图片，中等难度挑战
- **20×20难度**：画册后期21张图片，较高难度
- **25×25难度**：画册末尾15张图片，专家难度

### 难度递进示意

```
10×10 (21张) → 15×15 (18张) → 20×20 (21张) → 25×25 (15张)
第1-21张          第22-39张          第40-60张         第61-75张
```
