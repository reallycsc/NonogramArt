## 画册信息

| 属性 | 内容 |
| ---- | ---- |
| 序号 | 02 |
| 书架 | 人文历史 |
| 图片数 | 100 |
| 分块数/图 | 6 |
| 谜题数 | 610（常规600 + 专家10） |
| 难度配置 | 10×10~25×25 |
| 参考著作 | 斯塔夫里阿诺斯《全球通史》 |

## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：第一章《史前时代》、第二章《古代文明》，画册入门难度
- **15×15难度**：第三章《古典时代》全部、第四章前半部分，中等难度挑战
- **20×20难度**：第四章后半部分、第五章前半部分，较高难度
- **25×25难度**：第五章后半部分（第91-100号），专家难度

### 难度递进示意

```
10×10 (40张) → 15×15 (30张) → 20×20 (20张) → 25×25 (10张)
第1-2章          第3章+第4章前10   第4章后10+第5章前10  第5章后10
```

**像素图：**
- 命名规范：原图文件名 + `_pixel.png`
- 示例：`chapter1_01_africa_origin_pixel.png`

**关卡文件：**
- 目录：`data/puzzles/world_history/`
- 常规关卡命名：`{图片ID}_{分块索引}.json`
  - 示例：`chapter1_01_africa_origin_0.json` ~ `chapter1_01_africa_origin_5.json`
- 专家关卡命名：`{图片ID}_expert.json`
  - 示例：`chapter5_91_ww1_expert.json`
  - 仅第91-100号图片包含专家关卡
