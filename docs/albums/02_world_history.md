# 《世界通史》画册内容规划

## 画册信息

| 属性 | 内容 |
| ---- | ---- |
| 序号 | 02 |
| 书架 | 人文历史 |
| 图片数 | 100 |
| 分块数/图 | 6 |
| 谜题数 | 610（常规600 + 专家10） |
| 难度配置 | 10×10~25×25 |
| 参考著作 | 斯塔夫里阿诺斯《全球通史》 |

## 难度设计说明

中等难度范围，适合有一定数织基础的玩家

### 难度分布策略

- **10×10难度**：画册前31张图片，入门难度
- **15×15难度**：画册中间27张图片，中等难度挑战
- **20×20难度**：画册后期32张图片，较高难度
- **25×25难度**：画册末尾10张图片，专家难度

### 难度递进示意

```
10×10 (31张) → 15×15 (27张) → 20×20 (32张) → 25×25 (10张)
第1-31张          第32-58张          第59-90张         第91-100张
```
